# API Definitions

This directory contains all API definitions for the Tiny URL Shortener project.

## Structure

- **`openapi/`** - REST API specifications (OpenAPI/Swagger)
- **`proto/`** - gRPC Protocol Buffer definitions

## Protocol Buffers

Generate Go code from proto files:
```bash
make proto
```
